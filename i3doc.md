# Настройка тайлового оконного менеджера i3

## Простейшее использование
По умолчанию терминал можно запустить командой `$mod + Enter`, где (опять же по умолчанию) `$mod`  - клавиша `Alt`. При запуске терминал заполнит все возможное пространство.
### Перемещение по открытым окнам
Если открыть второе окно, оно будет размещено справа от текущего, если экран достаточно широкий, иначе - снизу.

Чтобы перемещаться между окнами, можно использовать комбинации, схожие с командами перемещения в `vim`. По умолчанию для перемещения влево - `$mod + j`, вправо - `$mod + ;`, вверх - `$mod + l`, вниз - `$mod + k`. Кроме того, можно использовать аналогичные комбинации со стрелками.

Если мы хотим изменить ориентацию разделения экрана, можно использовать комбинации `$mod + v` и `$mod + h`, первая - для вертикального разделения, вторая - для горизонтального разделения.
