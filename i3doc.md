# Настройка тайлового оконного менеджера i3

## Простейшее использование
По умолчанию терминал можно запустить командой `$mod + Enter`, где (опять же по умолчанию) `$mod`  - клавиша `Alt`. При запуске терминал заполнит все возможное пространство.
### Перемещение по открытым окнам
Если открыть второе окно, оно будет размещено справа от текущего, если экран достаточно широкий, иначе - снизу.
